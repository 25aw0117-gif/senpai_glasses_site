## 1. 概要
- **プロジェクト形態**: 前期課題（講師を依頼主とした制作）
- **担当ロール**: 企画、UI/UX設計、フロントエンド実装
- **制作期間**: 2024.05 ~ 2024.08（約3か月）

### サイト制作の目的
1. **情報の透明化**: 在校生の視点から見た「リアルな学生生活」を発信し、入学希望者の理解を深める。
2. **ミスマッチの解消**: 入学後のギャップを減らすことで、学生の意欲維持および退学率の低減に繋げる。
3. **志願者数の向上**: 信頼できる情報提供を通じて、学校の魅力を伝え、入学検討者を増やす。

### ターゲット (Target)
- 日本語能力試験N2以上を保有している外国人留学生。
- Webに興味はあるが、自分の希望職種は決めていな。

### コンセプト (Concept)
**「先輩のメガネ（Senpai's Glasses）」**
- **課題（視界の不透明さ）**: 日本での生活に不安や「見えづらさ」を感じている留学生（ターゲット）。
- **解決策（メガネを貸す）**: 先輩のリアルな体験談という「レンズ」を通すことで、学校生活を鮮明にイメージさせる。
- **成果（鮮明な視界）**: 正確な情報の習得と深い理解を促し、将来への視界をクリアにする。

## 3. このプロジェクトを通じて学んだこと (What i learned from the project)
- **ターゲットの本質的なニーズ把握**: 単なる情報提供ではなく、留学生特有の「油断」や「不安」を分析し、それに寄り添うコンテンツ設計の重要性を学びました。
- **コンセプトの視覚化**: 「メガネ」という抽象的なコンセプトをUIデザイン（直感的な情報の整理）に落とし込むプロセスを経験しました。

---

## 4. 解決した問題 (Problem Solving)
### 課題：情報の不足による不安とミスマッチ
- **問題点**: 公式サイトの情報だけでは、実際の授業の雰囲気や生活実態が伝わりにくい。
- **改善案**: 先輩へのインタビュー記事をメインコンテンツとし、具体的なエピソードを掲載。
